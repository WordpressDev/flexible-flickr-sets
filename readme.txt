== Description ==

Uses a shortcode to embed a responsive Flickr slideshow  set in a post / page. Uses the FlexSlider2 jQuery library from WooThemes. Requires a Flickr API key. [Get a Flickr API key here](http://www.flickr.com/services/api/keys/apply/ "Flickr API Key") 

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `flexible-flickr-sets` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Enter your Flickr API key
4. Place shortcode with the set ID from Flickr in a post / page.
